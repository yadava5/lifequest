# LifeQuest Desktop

Tauri 2 + Vite + React desktop client for LifeQuest.

## Development

```bash
npm install
npm run dev
```

For the native desktop shell:

```bash
npm run tauri dev
```

## Scripts

- `npm run dev` - Vite dev server
- `npm run tauri dev` - run the native shell
- `npm run build` - typecheck + Vite build
- `npm run lint` - lint the desktop app
